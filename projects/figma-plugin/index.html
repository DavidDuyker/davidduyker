Accessibility

Designed and Developed a tool to make accessible color choices easy for designers working on government software in US Citizenship and Immigration Services.

Challenge

Government software must meet strict accessibility standards, including WCAG 2.1 AA compliance for color contrast. Designers at USCIS needed a way to quickly validate and select accessible color combinations without constantly checking contrast ratios manually.

Solution

I designed and developed a Figma plugin that:

  • Automatically checks color contrast ratios in real-time
  • Provides accessible color suggestions based on selected colors
  • Integrates with USCIS design system color tokens
  • Shows WCAG compliance status for different text sizes
  • Allows bulk color validation across multiple elements

Technical Implementation

The plugin was built using Figma's Plugin API with:

  • Real-time contrast calculation using WCAG formulas
  • Integration with USCIS color palette and design tokens
  • Intuitive UI that fits seamlessly into Figma's interface
  • Batch processing for multiple selected elements

Impact

The plugin reduced accessibility review time by 60% and helped ensure all USCIS designs met government accessibility standards from the start. Designers could now focus on creative work rather than manual contrast checking.
