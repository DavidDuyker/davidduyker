Executive Summary

We founded the nZero Data-Download Tools on research sessions with sustainability experts and scientists; We developed these tools to help practitioners assemble their real time emissions and consumption data in to focused datasets for streamlined reporting. These tools drove 200% user engagement and quickly became the most used feature in the core product.
